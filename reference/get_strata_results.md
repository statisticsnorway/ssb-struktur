# Get strata results Get a table for results in each stratum for a rate model

Get strata results Get a table for results in each stratum for a rate
model

## Usage

``` r
get_strata_results(data, x = NULL, y = NULL)
```

## Arguments

- data:

  Population data frame with additional variables from rate_model output

- x:

  Name of the explanatory variable

- y:

  Name of the statistic variable

## Value

Table with strata results
