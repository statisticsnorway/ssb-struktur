# A function that shifts the elements in a vector one step to the right, the last element disappears while x_1 is inserted as the first element

A function that shifts the elements in a vector one step to the right,
the last element disappears while x_1 is inserted as the first element

## Usage

``` r
displace(x, x_1 = 0)
```

## Arguments

- x:

  A vector of numbers (NA's are allowed)

- x_1:

  A single number (NA are allowed)

## Value

A vector
