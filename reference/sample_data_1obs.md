# Sample test data (1 obs strata)

Test sample data for using in struktuR package. Contains a stratum with
1 observation for testing

## Usage

``` r
sample_data_1obs
```

## Format

A data frame with 1001 rows and variables:

- id:

  Identification variable

- employees:

  Number of employees

- employees_f:

  Number of female employees

- employees_m:

  Number of male employees

- turnover:

  Amount of turnover of the company (in 000 NOK)

- size:

  Category size of the company in terms of turnover (small, mid, big)

- industry:

  The activity group for which the business works in

- job_vacancies:

  The number of job vacancies in the company.

- sick_days:

  The number of sick leave days recorded for the company.

- sick_days_f:

  The number of sick leave days recorded for female employees for the
  company.

- sick_days_m:

  The number of sick leave days recorded for male employees for the
  company.
