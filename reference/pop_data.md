# Population test data

Test population data for using in struktuR package (synthetic data).

## Usage

``` r
pop_data
```

## Format

A data frame with 10000 rows and 7 variables:

- id:

  identification variable

- employees:

  number of employees

- employees_f:

  number of female employees

- employees_m:

  number of male employees

- turnover:

  amount of turnover of the company in NOK

- size:

  category size of the company in terms of turnover (small, mid, large)

- industry:

  the activity group for which the business works in
