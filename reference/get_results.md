# Get estimates Get estimates for groups (or strata) from rate model output

Get estimates Get estimates for groups (or strata) from rate model
output

## Usage

``` r
get_results(data, x = NULL, y = NULL, group = NULL)
```

## Arguments

- data:

  Population data frame with additional variables from rate_model output

- x:

  Name of the explanatory variable.

- y:

  Name of the statistic variable to estimate.

- group:

  Name of variable(s) for using for groups.

## Value

Table with strata or group results
